# Open Souce Software

Open source software allows collabrative development.

- GitHub is one of the most populer platforms for open source development.
- GitHub provides repositories, issue tracking, pull requests, and collaboration tools that make it easy for developers worldwide to contribute to open-source projects.
- Git is a version control system that tracks changes in files, while GitHub is a cloud-based platform that hosts Git repositories.

Some important terms related to GitHub:
1) Repository: A repository (repo) is a storage location on GitHub that contains a project's source code, documentation, and related files.
2) Forking: Forking means creating a personal copy of another user's repository so that you can make changes without affecting the original project.
3) Pull Request: A pull request is a request to merge changes from a forked or separate branch into the main project repository after review.
